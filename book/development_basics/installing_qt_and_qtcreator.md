# Встановлення Qt та QtCreator

Для початку нам необхідно встановити усе необхідне для роботи. Ми будемо використовувати кросплатформний засіб розробки Qt5 у складі якого маємо компілятор MinGW (тільки для Windows) та середовище для розробки QtCreator:
    для Linux - [download.qt.io/official_releases/qt/5.7/5.7.0/qt-opensource-linux-x64-5.7.0.run](download.qt.io/official_releases/qt/5.7/5.7.0/qt-opensource-linux-x64-5.7.0.run)
    для Windows - [download.qt.io/official_releases/qt/5.7/5.7.0/qt-opensource-windows-x86-mingw530-5.7.0.exe](download.qt.io/official_releases/qt/5.7/5.7.0/qt-opensource-windows-x86-mingw530-5.7.0.exe)

Якщо ви використовуєте Linux - встановіть компілятор С++ gcc (якщо у вас в системі він відсутній).

## Встановлення QtCreator та компілятора MinGW.

Скачайте інсталяційний файл та запустіть інсталяцію. Для встановлення немає необхідності реєструвати Qt Account. Для того, щоб продовжити без реєстрації просто натисніть Skip.

![](https://media.githubusercontent.com/media/PLLUG/CPPQT-Roadmap/master/storage/book/development_basics/installing_qt_and_qtcreator/SkipAccount.png)

	Для Windows нам також необхідно встановити компілятор MinGW. Для встановлення MinGW, переконайтеся, що його обрано у списку встаковлення (див. наступний малюнок):

![](https://media.githubusercontent.com/media/PLLUG/CPPQT-Roadmap/master/storage/book/development_basics/installing_qt_and_qtcreator/MinGW.png)
