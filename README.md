# CPPQT-Roadmap

Це можливість послідовно крок за кроком освоїти мову С++ та кросплатформний інструментарій розробки Qt5. У рамках PLLUG C++/Qt Roadmap відбуваються регулярні лекції та доповіді, які супровождуються матералами для самостійної роботи. Тут зібрані навчальні матеріали до PLLUG C++/Qt Roadmap.
